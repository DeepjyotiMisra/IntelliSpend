## ğŸ”‘ Key Concepts

### 1. **Embeddings**
- Text â†’ Vector (384 dimensions)
- Captures semantic meaning
- Similar texts = similar vectors

### 2. **FAISS Index**
- Pre-computed vector database
- Fast similarity search (milliseconds)
- Stores merchant descriptions as vectors

### 3. **Cosine Similarity**
- Measures angle between vectors
- Range: 0 to 1 (1 = identical)
- Used to find similar merchants

### 4. **Agent Tools**
- Functions agents can call
- Bridge between agents and utilities
- Return structured results

### 5. **RAG (Retrieval-Augmented Generation)**
- Retrieve relevant context first
- Use context for classification
- Future: LLM reasoning for edge cases

### 6. **Merchant Seed Expansion**
- Multiple transaction patterns per merchant
- Improves matching accuracy
- 159 patterns vs 26 (6x increase)
